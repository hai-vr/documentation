# Testing Program

Hello,

You have been directed to this page to participate in a testing program. This is only valid if the message you have received is directly

but you will find privacy considerations below:

### Privacy considerations

- Although your Steam Community name and details are not shown anywhere in the UI, your SteamID may become visible to other users in logs,
  connection attempts, or through reverse-engineering. This means your Steam Community name is not anonymous, even if it is never shown.
- The application may perform HTTP/S requests to URLs provided by other users, for instance, to load images or to load 3D objects.
  This may compromise your IP address.
- This program does not read any file, process, or other information from your computer network. Exceptions are our own application files themselves,
  and obvious API access such as OpenVR API access.
- The SDK of this program may ask the program to load files that were generated by the SDK.

Per the information above, by participating in this program, you understand that:
- other users may effectively discover your SteamID.
- other users may be able to discover your IP address.
