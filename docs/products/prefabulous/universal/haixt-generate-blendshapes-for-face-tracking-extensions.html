HaiXT Generate Blendshapes for Face Tracking Extensions

üåä Any Platform

If you have an avatar that already supports face tracking, this will generate additional non-standard blendshapes for use with HaiXT Face Tracking Extensions.

When your eyes are closed, the shape of your eyes will change when you're smiling.

warning

If you use FaceTra Shape Creator, you do not need this component as this feature already exists within FaceTra Shape Creator.

However, on the VRChat platform, you will still need the HaiXT Face Tracking Extensions component.

How to use‚Äã

To use this component:

  • Add a "PA HaiXT Generate Blendshapes for Face Tracking Extensions" component anywhere in your avatar.
  • Choose which blendshapes close the eyes with a smiling shape.

On the VRChat platform, you will need to add a HaiXT Face Tracking Extensions component to create the animator layers.

Versions‚Äã

  • 1.5.0: Added.

Classification: This component is application-agnostic.
