VRChatクレジット計算機（実験的）

VRChatクレジットには、単一の換算レートはありません。VRChatクレジットの価値は、お金を使うユーザーであるか、お金を稼ごうとする販売者であるか、あるいは販売から手数料を得ようとするVRChat社自身であるかによって異なります。

このページでは、関係各者間のVRChatクレジットの認識される価値を比較できます。

これはインタラクティブなスプレッドシート／換算表です。以下のいずれの欄も編集できます。

最善を尽くしましたが、不正確な部分があるかもしれません。ご自身でも十分にご確認ください。

数式​

ソースコードで使用されている数式を確認できます。

const c = {
    // Only the ratio between usdPaidByUser and vrchatCreditsGranted must remain correct
    usdPaidByUser: 10,
    vrchatCreditsGranted: 1200,
    // Other constants
    STEAM_OR_APPSTORE_REVENUE_SHARE: 0.3,
    VRCHAT_PLUS_MONTHLY_SUBSCRIPTION_PRICE_USD: 10,
    VRCHAT_PLUS_YEARLY_SUBSCRIPTION_PRICE_USD: 100,
    // Source used for fees: https://creators.vrchat.com/economy/payout/#fees
    PURCHASE_FEE: 0.4, // Note: PURCHASE_FEE must be higher or equal to STEAM_OR_APPSTORE_REVENUE_SHARE.
    TRANSACTION_FEE: 0.153,
    PAYOUT_FEE: 0.015,
    // Source used for minimum price: https://hello.vrchat.com/avatar-marketplace
    VRCHAT_AVATAR_MARKETPLACE_MINIMUM_PRICE_CREDITS: 1200,
};
// Credit purchase
const usdPaidByUser = c.usdPaidByUser;
const vrchatCreditsGranted = c.vrchatCreditsGranted;
const usdEarnedBySteam = c.STEAM_OR_APPSTORE_REVENUE_SHARE * c.usdPaidByUser;
const usdCreatorEconomy = (1 - c.PURCHASE_FEE) * c.usdPaidByUser;
// Sales and payout
const usdEarnedByVRChat = (c.usdPaidByUser - usdEarnedBySteam - usdCreatorEconomy) + c.TRANSACTION_FEE * usdCreatorEconomy;
const vrchatCreditsPayout = (1 - c.TRANSACTION_FEE) * c.vrchatCreditsGranted;
const usdPaymentProcessor = (usdCreatorEconomy - c.TRANSACTION_FEE * usdCreatorEconomy) * c.PAYOUT_FEE;
const usdPaidOutToCreator = usdCreatorEconomy - c.TRANSACTION_FEE * usdCreatorEconomy - usdPaymentProcessor;
// Estimations
const avatarsSoldAtMinPrice = c.vrchatCreditsGranted / c.VRCHAT_AVATAR_MARKETPLACE_MINIMUM_PRICE_CREDITS;
const vrcPlusMonthlySubs = usdEarnedByVRChat / ((1 - c.STEAM_OR_APPSTORE_REVENUE_SHARE) * c.VRCHAT_PLUS_MONTHLY_SUBSCRIPTION_PRICE_USD);
const vrcPlusYearlySubs = usdEarnedByVRChat / ((1 - c.STEAM_OR_APPSTORE_REVENUE_SHARE) * c.VRCHAT_PLUS_YEARLY_SUBSCRIPTION_PRICE_USD);
const employeesAt100kPerYear = usdEarnedByVRChat / 100_000;

収益分配とクレジット支払い率​

クリエイターエコノミーの収益分配に関する情報は、それらの計算には使用されません。検証のみに役立ちます。

30%/50%/20%の収益分配は、手数料を適用することで自然と導き出される割合です。あるいは、もっと可能性が高いのは、この特定の収益分配を得るために、いくつか手数料が選択されたということです。

同様に、200 Credits ≈ 1 USD と記載されているcredit payout rateは、ユーザーがクレジット購入のために最初に支払った金額から手数料が差し引かれた結果です。

Steam/Meta/Google/Appleの収益分配​

Steam、Meta、Google、Appleそれぞれの収益分配率は異なり、アプリによって[the total revenue generated by the app]、または[during the current year]、もしくはアプリのライフタイム、あるいは定期課金による継続課金であるかどうかに応じて変化する場合もあります。

さらに、VRChatとそれらのプラットフォーム間の商業取引や契約の可能性については、私たちが知らないものも含まれていません。

VRChatは、プラットフォーム手数料を30％とドキュメントに記載しており(https://creators.vrchat.com/economy/payout/#revenue-split)、アプリストアの上限も30％のようです。

Steam、Meta、Google、Appleが30%を下回る手数料を適用する場合、VRChatの手数料の方が高くなる可能性があります。

不一致​

公式ドキュメントによると、取引手数料は15.3%であり、販売者に1,016.4 VRChat Creditsが送金されることになります。しかし、下記スクリーンショットによると、販売者が実際に受け取る額は1015であるようです。

これにより、VRChatの取引手数料が15.4166…%（=185/1200）か15.333…%（=186/1200）のどちらであるかが不明確になります。このページでは、これらの推定手数料のいずれかを使用している可能性があります。

VRChatクレジット​

ユーザーが10米ドル相当のクレジット（VAT除く）を購入した場合：

  • そのユーザーは1200 VRChat Creditsを獲得します。
  • 支払いのために60%を（元の金額6ドル）取っておく場合があります。
  • Steam、Meta、Google、Appleは、最大30％（元の価格3ドル）を受け取ります。
  • 最低10%（元の金額1ドル）がVRChatに支払われます。

ユーザーが1200 VRChat Credits を使うと、

  • 販売者は1,015 VRChatクレジットを取得します（元の価値：8.45833…ドル；支払額：5.075ドル）。
  • VRChatは185 VRChat Creditsを保持しています（元の価値：1.54166…ドル；支払い額：0.925ドル）。
  • これにより、VRChatの手数料は15.333…%（=186/1200）よりも15.4166…%（=185/1200）に近いことになります。
  • 購入手数料を含めると、VRChatの手数料はこれまで1.925ドル（1ドル＋0.925ドル）です。
  • もし販売者も他の人のストアでVRChatクレジットを使えば、VRChatは手数料を稼ぎ続けます。

販売者がペイアウトをリクエストすると、取得した1,015 VRChat Credits（ペイアウト額5.075ドル） は、以下のようになります。

  • 支払処理業者は0.076125ドルを受け取ります。
  • 売り手は4.998875ドルを得ます。

つまり（概算で）：

  • ユーザーが10米ドル支払うと
  • Steam/Meta/Google/Appleは3ドルずつ入手する
  • VRChatは1925ドルを獲得しました（支払いを請求できないセラーは含まれません）。
  • 支払処理業者は0.075ドルを受け取ります。
  • 売り手は5ドル手に入れる

言い換えれば、

  • ユーザーにとって、1,200 VRChat Creditsのpaidの価値は10米ドルです。
  • ユーザーが最初に支払った1200 VRChatクレジットの価値は、販売者にとって5ドルの支払いに相当します（販売者は1015 VRChatクレジットを受け取ります）。
  • VRChatの1,200 VRChatクレジットがentering circulationすると、2ドル相当になります。
  • これは主張されている収益分配と一致しているようです。

警告：クレジットの支払いレートは 200クレジット = 1 USD です。

  • この支払率は、販売後に販売者が受け取るVRChatクレジットの額に基づいており、1,200 VRChatクレジットの販売に対して1,015 VRChatクレジットです。
  • ユーザーが取引手数料支払う前に支払ったVRChatクレジットを考慮すると、ユーザーが1200 VRChatクレジットを使うと、売り手は5ドルを得ることになり、これは販売前240 VRChatクレジット＝売り手1米ドルに近くなります。

まずはドルで：

  • VRChatがCEから1,000,000ドルを稼いだとき：
    • ユーザーは5,000,000ドルを支払い、売り上げは2,500,000ドルでした。
  • CEから1,000,000ドルの売上があった場合：
    • ユーザーは200万ドルを支払い、VRChatは40万ドルを稼ぎました。
  • ユーザーがCEに1,000,000ドル支払った場合：
    • VRChatは20万ドルを稼いでいる可能性があり、販売者は50万ドルを稼いでいる可能性があります。

VRChat Plus​

  • 月額料金は、月額10ドルです。
    • Steamは最大30％（3ドル）です。
    • VRChatは最低70%（7ドル）になります。
  • 年間購読料は年間100ドルです。
    • Steamは最大30%（30ドル）です。
    • VRChatは最低70%（70ドル）になります。

VRChat PlusとVRChatクレジットの比較​

その月の利用料金は月額10ドルです。

  • VRChatの月額課金は、1ヶ月あたり7ドルの収益をもたらします。
  • VRChatで同じ金額を稼ぐには、約4375 VRChatクレジットが必要です。
  • 4,375 VRChatクレジットは、ユーザーに約36.46ドルかかります。

年間購読料は、その年のユーザー料金が100ドルです。

  • VRChatの年間購読で70ドルの収益を得ます。
  • VRChatで同額を稼ぐには、約43,750 VRChat Creditsが必要です。
  • 43,750 VRChat クレジットは、ユーザーに約364.58ドルかかります。

アバターマーケットプレイスとの比較​

アバターの最低価格は1200 VRChat Creditsです。

  • VRChat Plusの月額購読1ヶ月分は、VRChatで最低価格で販売されるアバター3.65体分に相当します。
  • 年間VRChat Plusサブスクリプションは、VRChatが最低価格でアバターを36.46個販売した場合と同等の金額に相当します。

また、

  • 最低価格で販売された1,000のアバターは、VRChatが同額を稼ぐために販売する必要があるVRChat Plusの月額購読期間274.29ヶ月分に相当します。
  • 最低価格で販売された1,000体のアバターは、VRChatが同額を稼ぐために販売する必要があるVRChat Plus年間購読数の合計27.43年に相当します。

*出典：アバターマーケットプレイス、FAQ、「アバター制作者が従わなければならない最低価格というものがありますか？"

仮説上の米国給与との比較​

従業員が年間10万ドルの収入があると仮定すると：

  • そのような従業員1名分の年間給与を得るには、VRChatがVRChat Plusの年間購読を合計1428年間販売する必要があります。
  • そのような従業員1人が、その年に同額を稼ぐために、VRChatで62,500,000 VRChat Creditsを購入するために、ユーザーから520,833ドルの支払いを必要とします。

従業員10名がそれぞれ年間10万ドルの給与を受け取ると仮定します。

  • VRChat Plus の年間サブスクリプションを14,285年間分販売して初めて、そのような従業員10人の年間給与に相当する額をVRChatが稼ぐことができる。
  • そのような従業員10人が同額を稼ぐには、ユーザーが6億2500万VRChatクレジットを購入するために5,208,333ドルを支払う必要があります。

年間5万ドルの平均給与を受け取る従業員100人の会社を想定すると：

  • その会社が同額を稼ぐには、VRChatがその年にVRChat Plusの年間購読を合計71,428年分販売する必要があります。
  • その会社が同額を稼ぐには、その年、ユーザーが31億2500万VRChatクレジットを購入するために2604万1667ドルを支払う必要があります。

補足事項​

この文書では、ペイアウト閾値を下回っているためペイアウトを要求できず、他の場所でも使わない低パフォーマンスのセラー、および残りのクレジットを使わずにプラットフォームを離れるユーザーの可能性は考慮されていません。

このような場合、実際には決して使用されないクレジットが存在する可能性があり、VRChatはクリエイターエコノミーにおいて通常は支払い用に留保されている金額を請求できる可能性があります。
