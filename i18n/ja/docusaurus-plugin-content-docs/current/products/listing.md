# リポジトリリスト

これらの製品の多くは、弊社のパッケージリポジトリリストを使用してインストールできます。
- このリンクを使用して **[リポジトリリストをインストール](vcc://vpm/addRepo?url=https://hai-vr.github.io/vpm-listing/index.json)** してください。

<br />
<br />

:::info
このリポジトリリストでは、**[ALCOM](https://vrc-get.anatawa12.com/alcom/) パッケージマネージャー**の使用を推奨しています。

一部のパッケージマネージャー（*VCC* など）は、不具合があることが知られており、アップデートを正常にインストールできない場合があります。
不具合のあるパッケージマネージャーは、自己責任で使用してください。詳細は、[こちらの記事（英語）](/updates/2024/12/23/p0)をお読みください。
:::
