## 3.0.0-beta.4

변경 사항:
- 안티앨리어싱(anti-aliasing)을 8에서 4로 변경했습니다.
- ALT 키에 대한 여러 옵션을 추가했습니다.
  - 원본 사진을 표시하는 옵션을 추가했습니다. 이제 이것이 기본값입니다.
  - ALT 키의 기능을 비활성화하는 옵션을 추가했습니다.
  - 하이라이트된 핫스팟(hotspots)에 대한 옵션이 체크박스에서 두 가지 옵션으로 변경되었습니다.

수정 사항:
- 지원되지 않는 환경에서 Compute Shader가 실행되지 않도록 수정했습니다.

## 3.0.0-beta.3

수정 사항:
- Compute Shader에서 안티앨리어싱(anti-aliasing)에 더 큰 허용 오차(tolerance)가 필요하던 문제를 수정했습니다.
- 부적절한 RenderTexture 릴리스 호출을 수정했습니다.

## 3.0.0-beta.2

변경 사항:
- 더 밝아진 영역은 노란색으로, 더 어두워진 영역은 빨간색으로 표시하도록 변경되었습니다.
- 썸네일 생성 시 카메라에 안티앨리어싱(anti-aliasing)이 적용되도록 보장합니다.

## 3.0.0-beta.1

주요 변경 사항:
- 더 이상 Unity 2019를 지원하지 않습니다. Unity 2022 이상이 필요합니다.

최적화:
- 많은 수의 블렌드쉐이프를 포함하는 모델(700개의 블렌드쉐이프 테스트 완료)에서 UI가 더 빨라졌습니다.
- 렌더링 시간이 단축되었습니다.
  - Animation Mode의 가비지 컬렉션 특성으로 인해 Blendshape 창이 이미 열려 있는 경우 렌더링이 느려지던 문제가 있었습니다.
  - 이제 썸네일 렌더링에 Unity의 Animation Mode를 사용하지 않습니다.

기능:
- ALT 키를 누르면 블렌드쉐이프의 영향을 받는 영역이 노란색으로 표시됩니다.
- 체크박스 설정이 Editor 세션 간에 저장됩니다.
- 다국어 지원(Localize)을 추가했습니다.

변경 사항:
- Compute Shader를 지원하는 컴퓨터에서는 항상 Compute Shader를 사용합니다.
- 블렌드쉐이프 썸네일에 변화가 없는 경우, 썸네일이 더 어둡게 표시됩니다.
- "마우스 오버시 자동 업데이트(Auto Update On Focus)"가 기본적으로 꺼짐(OFF)으로 설정됩니다.
- 창 이름이 "BlendshapeViewer"에서 "Blendshape Viewer"로 변경되었습니다.

수정 사항:
- 자동 업데이트가 활성화된 상태에서 창에 포커스가 갈 때, 중복 새로고침을 방지하기 위해 업데이트가 지연되도록 수정되었습니다.
