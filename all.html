Skinned Mesh Constraint

Skinned Mesh Constraint creates a parent constraint that moves with the closest polygon of a Skinned Mesh Renderer.

🌊 Any Platform

FaceTra Shape Creator

FaceTra Shape Creator lets you create face tracking blendshapes on an avatar by morphing meshes.

🌊 Any Platform

Starmesh

Starmesh lets you iterate quickly on avatar gimmicks by deforming meshes directly in Unity.

🌊 Any Platform

Lightbox Viewer

Lightbox Viewer lets you simultaneously visualize changes in shader material settings under different lighting conditions.

🌊 Any Platform

Double Hip Tracker

Double Hip Tracker lets you use two hip trackers on VRChat. If one loses tracking, the other is used instead.

💬 Requires VRChat
Requires SteamVR

LetMeSee

LetMeSee lets you see your content in VR, with the Unity Editor in Edit Mode. The Scene tab is used as the camera viewpoint.

🌊 Any Platform

ResilienceVR

ResilienceVR is a standalone social VR app which focuses on experimentation and intimacy.

Animation Viewer

Animation Viewer lets you preview animations in the Project view browser.

🌊 Any Platform

Blendshape Viewer

Blendshape Viewer lets you visually browse blendshapes.

🌊 Any Platform

Property Finder

Property Finder lets you view the animatable properties of an object.

🌊 Any Platform

BlendTree Viewer

BlendTree Viewer lets you browse large blend trees in a compact view.

🌊 Any Platform

ComboGestureExpressions

ComboGestureExpressions lets you attach face expressions to hand gestures, and make it react to other Avatars 3.0 features.

💬 Requires VRChat

Constraint Track Animation Creator

Constraint Track Animation Creator lets you animate an object to slide along a track using constraints.

💬 Requires VRChat

Denormalized Avatar Exporter

Denormalized Avatar Exporter lets you export avatars into VTubing apps, without the disruptive pre-processing steps.

VSeeFace
VNyan
Warudo

Expressions Menu Hierarchy Editor

Expressions Menu Hierarchy Editor lets you edit Avatars 3.0 menus as if they were objects in your scene hierarchy.

💬 Requires VRChat

IconGen

IconGen lets you capture icons from objects in your scene, and then decorate and crop them.

🌊 Any Platform

IconGen Thumbnail

IconGen Thumbnail lets you take thumbnails in Play Mode, when the avatar is playing an animation.

🌊 Any Platform

Prefabulous

Prefabulous contains non-destructive components to automate invasive tasks in your personal avatar.

🌊 Any Platform

Recalculate Normals (Prefabulous)

Recalculate Normals improves the shading of SkinnedMeshRenderers by recalculating select blendshape normals.

🌊 Any Platform

Generate Twist Bones (Prefabulous)

Generate Twist Bones improves the elbow joints by automatically weight painting twist bones.

🌊 Any Platform

Vixen

Vixen provides another approach to create interactive elements and toggles on your avatar.

💬 Requires VRChat

Visual Expressions Editor

Visual Expressions Editor lets you edit face expression animations.

🌊 Any Platform

Animator As Code

Animator As Code is a Unity Editor API to generate Avatars 3.0 Animator layers and animations from a fluent builder syntax written in C#.

🌊 Any Platform

Modular Avatar As Code

Modular Avatar As Code contains facilities to automate the creation of Modular Avatar components using Animator As Code.

Cloth Transfer

Cloth Transfer lets you transfer cloth constraints across incompatible Unity versions.

🌊 Any Platform
