Blendshape Viewer

  • このページは日本語でご覧いただけます: 日本語
  • Cette page est disponible en français: Français

  • 本頁面有繁體中文版本: 繁體中文
영어 이외의 언어 번역은 Gemini 3 Flash에서 제공합니다. 또한, 애플리케이션에서 사용하는 로컬라이제이션 파일이 이 사용자 매뉴얼의 번역을 위한 추가 컨텍스트로 대규모 언어 모델에 제공됩니다.

3.0.0-beta.1​

주요 변경 사항:

  • 더 이상 Unity 2019를 지원하지 않습니다. Unity 2022 이상이 필요합니다.

최적화:

  • 많은 수의 블렌드쉐이프를 포함하는 모델(700개의 블렌드쉐이프 테스트 완료)에서 UI가 더 빨라졌습니다.
  • 렌더링 시간이 단축되었습니다.
    • Animation Mode의 가비지 컬렉션 특성으로 인해 Blendshape 창이 이미 열려 있는 경우 렌더링이 느려지던 문제가 있었습니다.
    • 이제 썸네일 렌더링에 Unity의 Animation Mode를 사용하지 않습니다.

기능:

  • ALT 키를 누르면 블렌드쉐이프의 영향을 받는 영역이 노란색으로 표시됩니다.
  • 체크박스 설정이 Editor 세션 간에 저장됩니다.
  • 다국어 지원(Localize)을 추가했습니다.

변경 사항:

  • Compute Shader를 지원하는 컴퓨터에서는 항상 Compute Shader를 사용합니다.
  • 블렌드쉐이프 썸네일에 변화가 없는 경우, 썸네일이 더 어둡게 표시됩니다.
  • "마우스 오버시 자동 업데이트(Auto Update On Focus)"가 기본적으로 꺼짐(OFF)으로 설정됩니다.
  • 창 이름이 "BlendshapeViewer"에서 "Blendshape Viewer"로 변경되었습니다.

수정 사항:

  • 자동 업데이트가 활성화된 상태에서 창에 포커스가 갈 때, 중복 새로고침을 방지하기 위해 업데이트가 지연되도록 수정되었습니다.

2.1.2​

  • 검색 결과가 0개일 때 레이아웃 오류가 더 이상 발생하지 않도록 수정되었습니다.
  • 사용자가 실수로 검색 필드에 관련 없는 긴 내용을 붙여넣었을 때 에디터가 멈추는 현상을 방지합니다.
    • 검색 쿼리가 최대 100자로 제한되었습니다.

2.1.1​

  • 메쉬가 선택되지 않았을 때 발생하는 오류를 수정했습니다.

2.1.0​

검색 및 클립보드 복사 추가:

  • 검색(Search) 필드를 추가했습니다.
  • 클립보드에 복사(Copy to Clipboard) 버튼을 추가했습니다.

2.0.3​

  • 아바타 업로드에 실패하는 문제를 수정했습니다.
    • 빌드 시 스크립트가 컴파일되도록 수정되었습니다.

2.0.2​

  • Blendshape Viewer의 첫 번째 공개 VCC 릴리스.
    • VCC 설치 시 Assets/에 있는 이전 버전의 설치 파일이 제거됩니다.
    • 이제 이 툴은 Packages/ 폴더에 위치합니다.
